### Вступление
Проект harmony core - это решение для автоматизации работы с музыкой в цифровом виде. Он представляет набор алгоритмов для решения простых музыкальных задач. 

### Цели
<p>Данный проект представляет набор пузыкальных примитивов, необходимых для представления нот и музыки в цифровом виде.</p>
<p>Также важнейшим аспектом проекта является возможность автоматической модификации музыкальных примитивов:</p>
<ul>
<li>Разрешение нот, интервалов и аккордов в заданной торальности.</li>
<li>Гармонизация мелодии (подбор баса, подбор средних голосов).</li>
<li>Гармонизация баса (подбор простой мелодии по басу).</li>
<li>Исправление группировки нот (правильный подбор длительности нот в зависимости от размера).</li>
</ul>

### Применение
<p>Библиотеку можно использовать в следующих областях:</p>
<ul>
<li>Для создания нотных редакторов.</li>
<li>В системах проверки работ учеников музыкальных учебных заведений.</li>
<li>Для создания ботов, выполняющих задания по музыке.</li>
</ul>

### Кому будет полезно?
<ul>
<li>Студентам музыкальных заведений, увлекающимся программированием.</li>
<li>Разработчикам музыкального софта.</li>
</ul>
